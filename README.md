# 101-AWS-LAB-SQS

## How to Use

### Send a Message

Enter a message in the text box.
Click "📩 Send to SQS".
A confirmation will be displayed.

### Receive Messages

Click "📥 Receive Messages".
Messages from the queue will be displayed and removed after processing.

### Refresh the Page

Click "🔄 Refresh Page" to update the queue count without resubmitting forms.
